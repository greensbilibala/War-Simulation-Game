# 美伊局势模拟器

网页端策略小游戏：**美伊局势**背景，幽默诙谐但基于现实。选择美国（进攻方）或伊朗（防守方），通过军费、兵力、股市、原油、舆论（支持率）和每日策略与随机事件决出胜负；事件中融入海峡封锁、石油禁运、中国调停等真实议题。

## 玩法

- **选边**：美国（军费足、看股市与国内支持率）或伊朗（靠原油收入、海峡牌、可拉中国调停）。
- **每日**：先执行**一种策略**（如美国：增兵/空袭/制裁/外交/稳股市；伊朗：增兵/封锁海峡/导弹袭击/外交/争取中国），再点击「下一日」进入结算并触发**随机事件**。
- **数据**：双方各有军费、兵力、支持率；全球有原油价格、美股指数；另有「中国」态度（-10 亲美 ~ +10 亲伊），影响事件与调停。
- **胜负**：任一方支持率归零或军力耗尽则对方胜；若约 30 天内未决出，按局势与实力综合判定。
- **存档/读档**：浏览器 localStorage，刷新后可读档继续。

## 运行

用浏览器直接打开 `index.html`，或本地静态服务：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 项目结构

```
战争模拟器/
├── index.html      # 选边 + 主界面
├── css/style.css   # 样式（双阵营、全球数据）
├── js/
│   ├── game.js    # 美伊双边状态、全球原油/股市、策略与胜负
│   ├── events.js  # 美伊+中国现实梗随机事件
│   └── ui.js      # 选边、策略、事件弹窗、存档
└── README.md
```

## 扩展建议

- 在 `events.js` 的 `GameEvents` 中增加更多现实梗事件。
- 在 `game.js` 中细化中国态度对事件选项或结局的影响。
- 增加多档位存档或难度选项。
